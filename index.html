Setup Ruby DevKit Windows

Apr 17th, 2014

Môi trường

  • Windows 8.1
  • Ruby DevKit 4.7.2

Thực hiện

  • Truy cập http://rubyinstaller.org/
  • Vào mục http://rubyinstaller.org/downloads/ xem và download phần Development Kit (phiên bản 4.7.2 cho 64bits tên file DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe)
  • Chạy tập tin DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe và làm theo hướng dẫn (chọn thư mục cài đặt C:\abc\RubyDevKit)
  • Thêm đường dẫn “C:\abc\RubyDevKit\bin” vào biến môi trường (Environment variables) “Path”
