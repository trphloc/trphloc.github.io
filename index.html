Diskpart

Jan 15th, 2015

Công cụ hỗ trợ xử lý đĩa cứng

  • Thực thi lệnh diskpart trong cửa sổ command
  • Một số lệnh thông dụng
    • List disk: hiển thị thông tin các ổ đĩa
    • Select disk … : chọn ổ đĩa hiện hành (các thao tác sẽ thực thi trên ổ đĩa này)
    • Select partition …: chọn partition hiện hành
    • Select volume …: chọn volume hiện hành
    • detail disk: thông tin ổ đĩa
    • clean: xóa toàn bộ thông tin ổ đĩa
    • create partition …: tạo partition mới

Xử lý “GPT Protective Partition” trên windows xp

  • Chọn phím Windows + R
  • Gõ lệnh diskpart
  • Thực thi lệnh List disk
  • Chọn ổ đĩa cần xử lý Select disk …
  • Thực thi lệnh clean => sẽ xóa toàn bộ dữ liệu trên ổ đĩa
  • Thực thi lệnh create partition primary
  • Thực thi lệnh assign G (G là ổ đĩa map với ổ cứng)
  • Tài liệu tham khảo
    • http://support.microsoft.com/kb/300415
    • http://technet.microsoft.com/en-us/library/cc766465%28v=ws.10%29.aspx

Connect to SQL Server Using Oracle SQL Developer

Jan 15th, 2015

Sử dụng Oracle SQL developer kết nối tới SQL Server

Download tập tin jTDS

  • http://sourceforge.net/projects/jtds/files/
  • Sử dụng phiên bản mới nhất
  • Giải nén vào thư mục abc_xyz

Cấu hình Oracle SQL Developer

  • Mở cấu hình trong Oracle SQL Developer
    • Windows: menu Tools => Preferences
    • Mac: Oracle SQL Developer => Preferences
  • Chọn Database => Third party JDBC drivers
  • Chọn add entry

  • Chọn tập tin jtds-x.x.jar
  • Lưu và tắt preferences
  • Khởi động lại Oracle SQL developer

Kết nối

  • Chọn new connection
  • Chọn SQLserver và điền thông tin

  • Test và kết nối

Tài liệu tham khảo

  • http://kentgraziano.com/2013/01/14/tech-tip-connect-to-sql-server-using-oracle-sql-developer/
