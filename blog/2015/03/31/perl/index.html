ActivePerl

Cài đặt

  • Tải tại đây (http://www.activestate.com/activeperl/downloads) và cài đặt theo chỉ dẫn
  • Thêm đường dẫn vào biến PATH (sử dụng file .profile)
  • Tạo Symbolic link tới /usr/bin/perl và ppm

  • Cài đặt và sử dụng 1 phiên bản perl cụ thể
